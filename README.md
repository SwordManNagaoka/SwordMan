# 概要
+ ソードマン長岡が迫りくる敵を切り倒すランアクションゲームです
+ アンドロイド(5.0以上)をターゲットに開発しています
+ Windowsにも対応予定です
## 操作方法
+ ジャンプ:画面左半分をタップ
+ 攻撃：画面右半分をタップ
+ ジャンプ攻撃：ジャンプ中に画面左半分をタップ
+ ポーズ：右上のアイコンをタップ
## 制作環境
+ 使用したコンパイラー：VisualStudio2017
+ 使用言語：C++17(アンドロイド環境では一部機能が使用不可)
+ 使用した外部エディター：Tiled
+ 使用した外部ライブラリ: DXライブラリ http://dxlib.o.oo7.jp/
